# Blog Pessoal - Landing Page

Uma página simples e responsiva criada com HTML e CSS para exibição de perfil e tópicos de interesse para o trabalho da disciplina de Web 1.

## Como usar
1. Salve o código principal em um arquivo chamado `index.html`.
2. Abra o arquivo em qualquer navegador web.

## Tecnologias
- HTML5
- CSS3
